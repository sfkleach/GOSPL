Welcome to the GOSPL
====================

The GOSPL is a public collection of POPLOG source code that is 
reviewed and maintained by Stephen Leach.

Contents
--------

gospl           The master GOSPL archive itself.
output          Output folder for builds.
pending         Incoming contributions.
releases        Folder of public releases.
retired         Old contributions that have been removed.
tools           The development & build tools.

For GOSPL Users
---------------

All you need is the gospl directory. For detailed instructions see the
gospl/README.txt file.


For GOSPL developers
--------------------

You need to checkout the whole trunk. For detailed instructions read the
tools/README.txt file.
